    * Two Web Servers
    * Load Balancer
    * Virtual Network
    * Network Security Group
      - Inbound Security Rules
      
The NetworkDiagram-ELK expanded further into implementing an additional Virtual Network, with a server that will operate as the ELK Stack server.
